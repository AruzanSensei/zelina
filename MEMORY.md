# MEMORY.md - Curated Wisdom & Workflow

## Workflow Habits
- **URL Mapping**: 
  - Proyek di root (`zanxa.site/...`) dipresentasikan sebagai `https://zanxa.site/...`.
  - Proyek di folder `apps/` dipresentasikan sebagai subdomain `https://[folder-name].zanxa.site/`.
- **Git Protocol**: 
  - Tidak ada *auto-push* ke GitHub.
  - Push hanya boleh dilakukan setelah konfirmasi eksplisit dari user ("silakan push" atau konfirmasi bahwa web sudah final).
  - Jika revisi masih berjalan, simpan perubahan lokal saja.

## ⚠️ Mandatory Pre-Response Protocol
Read all instructions in this file (and other relevant MD files) before responding to chats or executing instructions.
