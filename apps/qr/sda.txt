Berikut versi update poin nomor 1 yang sudah diperluas berdasarkan struktur dashboard dan semua saran menu tambahan diberi label `(optional)`.

# 1. Tambahkan halaman admin

Pastikan admin panel bisa melakukan CREATE, READ, UPDATE, DELETE (CRUD) data pada database yang akan diperbarui (lihat nomor 6).

Pastikan UI berbentuk seperti dashboard modern dengan sidebar menu di sisi kiri.

## Sidebar Menu

### 1.1 Dashboard

Halaman overview utama admin.

Fitur:

* Total produk
* Statistik scan QR
* Produk terbaru
* Aktivitas terbaru
* Quick action

---

### 1.2 Products

Menampilkan semua produk dan detail produk.

Fitur:

* Create product
* Read product
* Update product
* Delete product
* Search produk
* Filter produk
* Pagination
* Detail produk lengkap
* Upload multiple images

Field produk:

* Nama produk
* Tipe / kode
* Nomor seri
* Tahun pembuatan
* Input
* Output
* Frekuensi
* Jumlah soket
* Range daya
* Soft Fuse Protection
* Hard Fuse Protection
* Ground Output
* Tambahan / optional
* Gambar depan
* Gambar kanan
* Gambar kiri
* Gambar belakang

Submenu:

* All Products
* Add Product
* Archived Products (optional)
* Product Categories (optional)

---

### 1.3 QR Codes

Halaman khusus untuk manajemen QR code.

Fitur:

* Generate QR otomatis
* single/ multiple Download QR
* Preview QR
* Mapping QR → produk
* QR status checker
* Bulk print QR (optional)

---

### 1.4 Scan Analytics

Fitur:

* Browser pengguna
* Lokasi scan
* Invalid scan detection
* Grafik statistik

(optional)

## UI / UX Requirement

* Responsive dashboard layout
* Sidebar collapsible
* Dark text table style
* Corporate techy bold style
* Dominasi warna:

  * 60% putih
  * 30% hitam
  * 10% hijau
* Menggunakan ETS logo:
  assets/ets-logo.png
* Modern table UI
* Search & filtering cepat
* Toast notification
* Modal confirmation untuk delete
* Skeleton loading
* Empty state UI
* Consistent spacing & typography
