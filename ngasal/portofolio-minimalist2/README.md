# portofolio-minimalist2

Versi kedua dari eksperimen portfolio minimalist.

## Struktur file

- `index.html` → halaman utama
- `assets/css/style.css` → styling utama
- `assets/js/script.js` → data portfolio, menu mobile, dan animasi reveal

## Karakter desain

- white minimalist
- mobile-first
- responsif
- cocok untuk menampilkan banyak project
- lebih profesional dan meyakinkan untuk calon klien

## Cara buka

Buka `index.html` langsung di browser lokal.
